# 💌 Love Letter
![Love Letter Preview](img/background.png)

An interactive love letter website featuring an animated envelope, a smooth letter reveal, and a confetti celebration effect. This project was created to turn a simple message into a memorable digital experience using front-end web technologies.

## ✨ Features

* Animated envelope opening interaction
* Smooth letter reveal animation
* Confetti effect triggered when opening the letter
* Custom background imagery
* Custom embedded font for a unique appearance
* Responsive design for desktop and mobile devices
* Lightweight implementation with no frameworks or dependencies

## 🛠️ Technologies Used

* **HTML5** – Page structure and content
* **CSS3** – Styling, animations, and visual effects
* **JavaScript** – Interactive functionality and confetti animation
* **WOFF2 Custom Font** – Personalized typography

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/DanLeung85/Love-Letter.git
```

### Open the Project

```bash
cd Love-Letter
```

Open `index.html` in your preferred web browser.

No installation, build tools, or dependencies are required.

## 🎨 Customization

You can personalize the project by:

* Editing the letter content in `index.html`
* Replacing the background image in the `img` folder
* Changing fonts and styling in `style.css`
* Adjusting animations and effects in `script.js`
* Modifying envelope colors, sizes, and transitions
